JDS Programming Language

Welcome to the official documentation for JDS. JDS is a compiled, statically-structured scripting language that compiles directly into x86_64 assembly (NASM) for Linux.

It features a clean syntax, standard mathematical operations with operator precedence, control flow statements, and built-in I/O capabilities. JDS also supports modular execution, allowing you to load and execute raw binary files directly into memory.

Usage & Compilation

The compiler requires your source files to have the .jds extension.

# Compile a standard program
./compiler main.jds

# This generates an ELF64 executable
./main

Syntax Basics

Comments

JDS supports single-line comments using double slashes.

// This is a comment and will be ignored by the lexer

Variables & Data Types

Variables are declared using the let keyword. In JDS, variables are implicitly 64-bit integers. You can declare them with or without an initial value.

let x = 10
let y
y = 20

Arithmetic Expressions

JDS supports standard arithmetic operations: addition (+), subtraction (-), multiplication (*), and division (/). Multiplication and division have a higher precedence than addition and subtraction.

let result = 10 + 5 * 2  // result is 20
let div = 20 / 4         // div is 5

Control Flow

If / Elif / Else Statements

Conditional logic uses standard blocks surrounded by curly braces {}. Supported comparison operators are ==, <, and >.

let a = 15

if a < 10 {
    print "Less than 10"
} elif a == 15 {
    print "Exactly 15"
} else {
    print "Greater than 10"
}

While Loops

Loops in JDS are implemented using the while keyword.

let count = 5
while count > 0 {
    print count
    count = count - 1
}

Input / Output & System

Printing

Use the print keyword to output text or variables. String literals support escape sequences like \n, \t, and \".

print "Hello, World!\n"
let score = 100
print score

Input

The input keyword reads an integer from standard input and stores it in a variable.

let user_input
input user_input
print "You entered:"
print user_input

Exit

You can forcefully terminate the program and return an exit code to the operating system using exit.

exit 0  // Exit with success code

Built-in Variables

argc is a built-in variable that holds the number of command-line arguments passed to the program.

print "Argument count:"
print argc

Modules & Exec

Defining a Module

If a file starts with the module keyword, it is compiled as a raw binary module instead of a standalone ELF executable. This is useful for writing dynamic code to be loaded later.

module
print "I am a dynamically loaded module!"
exit 0

Executing Binaries in Memory

The exec statement allows the program to read a raw binary file from disk, map it directly into executable memory, and call it. This acts as a highly efficient plugin or script execution system.

// Loads "script.bin" into memory and executes it
exec "script.bin"

Мова програмування JDS

Вітаємо в офіційній документації для JDS. JDS — це компільована скриптова мова зі статичною структурою, яка компілюється безпосередньо в асемблер x86_64 (NASM) для Linux.

Вона має чистий синтаксис, стандартні математичні операції з пріоритетом операторів, конструкції керування потоком та вбудовані можливості вводу/виводу. JDS також підтримує модульне виконання, що дозволяє завантажувати та виконувати сирі бінарні файли прямо в пам'яті.

Використання та компіляція

Компілятор вимагає, щоб ваші файли мали розширення .jds.

# Компіляція стандартної програми
./compiler main.jds

# Це згенерує виконуваний файл ELF64
./main

Основи синтаксису

Коментарі

JDS підтримує однорядкові коментарі, використовуючи подвійний слеш.

// Це коментар, він буде проігнорований лексером

Змінні та типи даних

Змінні оголошуються за допомогою ключового слова let. У JDS змінні є неявно 64-бітними цілими числами. Ви можете оголошувати їх з початковим значенням або без нього.

let x = 10
let y
y = 20

Арифметичні вирази

JDS підтримує стандартні арифметичні операції: додавання (+), віднімання (-), множення (*) та ділення (/). Множення та ділення мають вищий пріоритет, ніж додавання та віднімання.

let result = 10 + 5 * 2  // result дорівнює 20
let div = 20 / 4         // div дорівнює 5

Керування потоком

Конструкції If / Elif / Else

Логічні умови використовують стандартні блоки, оточені фігурними дужками {}. Підтримувані оператори порівняння: ==, < та >.

let a = 15

if a < 10 {
    print "Менше ніж 10"
} elif a == 15 {
    print "Рівно 15"
} else {
    print "Більше ніж 10"
}

Цикли While

Цикли в JDS реалізуються за допомогою ключового слова while.

let count = 5
while count > 0 {
    print count
    count = count - 1
}

Введення / Виведення та Система

Виведення (Print)

Використовуйте ключове слово print для виведення тексту або змінних. Рядкові літерали підтримують escape-послідовності, такі як \n, \t та \".

print "Привіт, світе!\n"
let score = 100
print score

Введення (Input)

Ключове слово input зчитує ціле число зі стандартного вводу і зберігає його у змінну.

let user_input
input user_input
print "Ви ввели:"
print user_input

Вихід (Exit)

Ви можете примусово завершити програму та повернути код завершення операційній системі за допомогою exit.

exit 0  // Завершення з кодом успіху

Вбудовані змінні

argc — це вбудована змінна, яка містить кількість аргументів командного рядка, переданих програмі.

print "Кількість аргументів:"
print argc

Модулі та Exec

Визначення модуля

Якщо файл починається з ключового слова module, він компілюється як сирий (raw) бінарний модуль замість самостійного ELF файлу. Це корисно для написання динамічного коду, який завантажується пізніше.

module
print "Я — динамічно завантажений модуль!"
exit 0

Виконання бінарних файлів у пам'яті

Інструкція exec дозволяє програмі зчитувати сирий бінарний файл з диска, відображати його безпосередньо у виконувану пам'ять (mmap) і викликати. Це працює як високоефективна система плагінів.

// Завантажує "script.bin" в пам'ять і виконує його
exec "script.bin"
